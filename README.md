# Street Quiz (ироничная викторина)

Небольшой стартовый шаблон мобильного приложения-викторины на React Native (Expo).

## Важно про тон и контент

Проект использует сарказм и иронию в формулировках, но без прямого унижения людей, оказавшихся в трудной жизненной ситуации. Рекомендуется:

- шутить над абсурдными ситуациями, а не над достоинством человека;
- избегать оскорбительных ярлыков;
- добавлять дисклеймер о том, что игра — художественная сатира.

## Что уже реализовано

- стартовый экран с названием и кнопкой начала;
- 5 вопросов с 4 вариантами ответов;
- подсчёт очков;
- экран результата и перезапуск викторины.

## Запуск

```bash
npm install
npm run start
```

Для веб-просмотра во время разработки:

```bash
npm run web
```

## Структура

- `App.tsx` — вся логика MVP викторины;
- `package.json` — скрипты запуска Expo;
- `tsconfig.json` / `babel.config.js` — базовая конфигурация TypeScript и Babel.
